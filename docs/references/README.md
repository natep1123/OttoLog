## How agents should use these

> Open every `.jpg` under `docs/references/workout-builder/` **or** `docs/references/query-builder/` (whichever matches the work) with the Read tool. Do not invent missing checklist files.

Reading a markdown `![](...)` link alone does **not** load pixels — open the image file.

## Maintaining accuracy

After a chrome pass: re-shoot lean slots → drop into the canonical folder with checklist names → update that folder’s README. Put one-offs in `archive/` only if worth keeping.
